4. 콘텐츠 그룹 태그<br>

ol 태그 : 순서 있음.<br>
ul 태그 : 순서 없음. 메뉴 만들때 활용 많이 함.<br><br>

ol 태그의 숫자매기기를  
로마자 소문자로 바꾸기 : ol태그에 type='i' 입력  
로마자 대문자로 바꾸기 :  ol태그에 type='I' 입력  
알파벳 소문자 순서로 바꾸기 : ol태그에 type='a' 입력  
알파벳 대문자 순서로 바꾸기 : ol태그에 type='A' 입력  
<br><br>
div태그 : 수많은 태그들과 컨텐츠들을 '묶는 태그'
