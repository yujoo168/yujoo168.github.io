만약 else if문이 많아진다면, 즉 세부적인 조건들이 많아진다면 그걸 다 보게 된다. 근데 꼭 그럴 필요가 없을 때도 존재. 그럴 때 switch문을 쓴다.
모든 조건을 다 훑지 않고, 조건이 충족되는 내용으로 바로 간다!

    var day;
    switch (new Date().getDay()) {
      case 0:
        day = 'sunday'
        break;
      case 1:
        day = 'mon'
        break;
      case 2:
        day = 'tue'
        break;
      case 3:
        day = 'wed'
        break;
    }
    document.write(day);

여기서
case 0:
case 1:
case 2:
과 같은 식으로 써서 해당하는 조건에 바로 가게 한다.
