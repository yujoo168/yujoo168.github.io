for문과 while문을 배울 것. 둘 다 반복문인듯?

    for (var i = 0; i < 10; i++) {
      document.write('for hello', '<br>');
    }

    var x = 0;
    while (x < 10) {
      document.write('while hello', '<br>');
      x += 1;
    }

for 안에 for 또 들어갈 수 있다.
